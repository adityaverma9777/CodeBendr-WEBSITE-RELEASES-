## Built On

### Visual Studio Code
CodeBendr is built on top of **Visual Studio Code**, the popular open-source code editor by Microsoft.

- **Editor Foundation**: VS Code provides the core editing experience
- **Extension API**: Enables our custom learning features
- **Cross-Platform**: Works on Windows, macOS, and Linux

Learn more at [code.visualstudio.com](https://code.visualstudio.com/)

### Open Source Technologies

We stand on the shoulders of giants. CodeBendr utilizes:
- **Node.js**: JavaScript runtime
- **Electron**: Desktop application framework
- **Monaco Editor**: The code editor that powers VS Code
- Various npm packages and open-source libraries
