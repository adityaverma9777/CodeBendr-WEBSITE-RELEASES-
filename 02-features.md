# Features

CodeBendr IDE comes packed with powerful features designed to enhance your learning experience.

## Built-In Learning System

### Structured Curriculum
- **Three Programming Languages**: HTML, Python, and Java
- **Progressive Learning Path**: Start from basics and build to advanced topics
- **Organized by Chapters**: Content divided into logical learning units

### Interactive Lessons
- **Rich Markdown Content**: Beautifully formatted lessons with code examples
- **Syntax Highlighting**: Code blocks with proper formatting
- **Progress Tracking**: Mark lessons as complete and track your journey

### Hands-On Assignments
- **Practice Problems**: Apply what you've learned
- **Integrated Editor**: Write code directly in the IDE
- **Solution Viewer**: Check solutions when you're stuck
- **Code Execution**: Run and test your code instantly

### Knowledge Assessment  
- **Built-In Quizzes**: Test your understanding after each lesson
- **Instant Feedback**: Know immediately if you passed
- **80% Pass Threshold**: Ensures solid understanding before advancing
- **Unlimited Attempts**: Keep trying until you master the concept

### Real-World Projects
- **Practical Applications**: Build actual projects
- **Starter Templates**: Get a head start with pre-configured files
- **File Management**: Open and edit project files seamlessly

## Custom Branding

- **CodeBendr Logo**: Custom branding throughout the IDE
- **Professional Interface**: Clean, focused learning environment
- **Activity Bar Integration**: Quick access to learning materials

## VS Code Foundation

Built on Visual Studio Code, you get:
- **Professional Code Editor**: Industry-standard editing experience
- **Extensions Support**: Add more functionality as needed
- **Git Integration**: Learn version control from day one
- **Debugging Tools**: Understand and fix your code
- **Terminal Access**: Full command-line integration
